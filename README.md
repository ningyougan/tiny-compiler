## Note

前端工程化中，社区中很多著名项目都依赖于`acron`的能力，比如：ESLint escheck Babel Vue.js 等。acron 是一个完全使用 JavaScript 实现的，小且快速的 JavaScript 解析器。学习编译器，会对这些常用工具理解更加深刻。
